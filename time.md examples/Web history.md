# Web history

Browser activity from the **Web History** export section. Each tab is a
separate embed with its own widget.

## Timeline

```timemd
view: web-history
tab: timeline
limit: 100
title: Recent visits
```

## Top domains

```timemd
view: web-history
tab: domains
title: Where I spent my browsing time
```

## Hourly activity

```timemd
view: web-history
tab: activity
title: Visits by hour
```

## Filter to a single browser

```timemd
view: web-history
browser: Arc
tab: timeline
limit: 50
title: Arc only
```
